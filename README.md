# Smart Attendance System

## Overview

Smart Attendance System is a mobile and web-based attendance management solution that uses Face Recognition technology to automate student attendance.

The system allows administrators and teachers to manage departments, classes, subjects, students, and attendance records through an easy-to-use interface.

## Features

* Face Recognition Attendance
* Manual Attendance Management
* Student Registration with Face Encoding
* Department Management
* Class Management
* Subject Management
* Attendance Reports
* Low Attendance Detection
* Weekly and Monthly Analytics
* Search by Roll Number
* Attendance History Tracking

## Technology Stack

### Frontend

* Flutter

### Backend

* Flask (Python)

### Database

* MySQL

### Face Recognition

* DeepFace
* OpenCV

## Installation

### Backend
